# Трекер привычек (Habit Tracker)

Мобильное приложение на **React Native + Expo Router + Redux Toolkit + TypeScript**
для отслеживания ежедневных привычек. У каждой привычки есть показатель
выполнения от 0 до 100.

## Стек и архитектура

- **Expo Router** — файловая навигация, вкладки в `app/(tabs)`
- **Redux Toolkit** — управление состоянием, один slice (`store/habitsSlice.ts`)
- **Typed hooks** — `useAppDispatch` / `useAppSelector` (`store/hooks.ts`)
- **TypeScript** — строгая типизация

## Структура проекта

```
app/
  _layout.tsx            # корневой layout, подключает Redux Provider
  (tabs)/
    _layout.tsx          # нижняя навигация (2 вкладки)
    index.tsx            # вкладка «Привычки»
    stats.tsx            # вкладка «Статистика»
components/
  HabitCard.tsx          # карточка привычки
  ProgressBar.tsx        # полоска прогресса
store/
  index.ts               # configureStore + типы RootState / AppDispatch
  hooks.ts               # типизированные хуки
  habitsSlice.ts         # slice: привычки + история + статистика
```

## Возможности

### Вкладка «Привычки»
- Список привычек: название, полоска прогресса, текущее значение (%)
- Кнопки управления для каждой привычки:
  - **Выполнить** — увеличивает прогресс (+20, до 100)
  - **Пропустить** — уменьшает прогресс (−20, до 0)
  - **Сброс** — обнуляет прогресс
- Добавление и удаление привычек

### Вкладка «Статистика»
- Общее количество выполнений привычек
- Количество привычек
- История всех действий (выполнено / пропущено / сброшено) с датой и временем

## Запуск

```bash
npm install
npm start          # затем 'i' для iOS, 'a' для Android, 'w' для web
```
